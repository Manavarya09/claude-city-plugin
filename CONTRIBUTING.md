# Contributing

Thanks for your interest! This repo is small — patches and ideas welcome.

## Quick start
```bash
git clone https://github.com/Manavarya09/claude-city-plugin
cd claude-city-plugin
npm install
```

## Running locally
See the README for the `/city` slash command and the local web app under `app/`.

## PR checklist
- [ ] Branch from `main`, named like `feat/...` or `fix/...`
- [ ] Lint passes
- [ ] No leftover `console.log`s
- [ ] PR description explains the *why*

## Reporting bugs
Open an issue with steps to reproduce. Security issues: see SECURITY.md.
